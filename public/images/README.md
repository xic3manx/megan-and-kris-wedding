# Image folder

Drop iPhone photos here. Recommended structure:

- `couple/` — photos of the two of you (engagement, candids, etc.)
  - `hero.jpg` — primary hero photo on Home page (landscape, ~1600x1000px ideal)
  - `og.jpg` — social/Open Graph preview image (1200x630)
- `venues/` — venue exterior/interior photos
  - `pelican-hill.jpg` — main Pelican Hill image for Itinerary
  - `mastros.jpg` — Mastro's Ocean Club for Itinerary
- `decor/` — flowers, snail illustrations, ornamental imagery

Filenames in lowercase with hyphens. The site references them as
`/images/couple/hero.jpg`, `/images/venues/pelican-hill.jpg`, etc.

For venues you don't have photos of, the page will gracefully fall back
to elegant placeholder treatments until you drop in real images.
